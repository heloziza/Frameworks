<h1>Frameworks</h1>

<h2>Express.js</h2>
<p>Esse framework teve a sua primeira versão lançada no dia 22 de Maio de 2010 e foi fundado por TJ Holowaychuk, porém contem mais desenvolvedores.<br>Sua linguagem de programação é JavaScript e oferece soluções para os seguintes casos:<ul><li>Gerenciar requisições de diferentes verbos HTTP em diferentes URLs.</li><li>Integrar "view engines" para inserir dados nos templates.</li><li>Definir as configurações comuns da aplicação web, como a porta a ser usada para conexão e a localização dos modelos que são usados para renderizar a resposta.</li><li>Adicionar novos processos de requisição por meio de "middleware" em qualquer ponto da "fila" de requisições.</li></ul></p>
